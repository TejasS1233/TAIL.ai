TAIL.ai — Dev Server

Backend: http://localhost:8002

This is a lightweight static fallback page served by the backend so the website is reachable while the full React + ML stack is brought up.

Available API endpoints (examples):

  • /api/v1/healthcheck
  • /api/v1/dev/token (POST)
  • /api/v1/scan (POST upload)

To run the full frontend and ML microservice, follow one of these options (recommended):

  1. Install Docker and run docker compose up --build to run the full stack in containers.
  2. Or install Python 3.11 and Node >=22.12 locally, then run the ML service and the client dev server:
cd ml_service
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python -m uvicorn app.main:app --host 127.0.0.1 --port 8003 --reload

cd ..\client
npm install
npm run dev
